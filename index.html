VTNutrition

Repository for VTNutrition app, listing nutrition information for VT dining halls

View project onGitHub

Not implemented yet

The daily calorie counter/planner is not implemented. We need icons and graphics.

Usage

Dining Halls tab

This tab features a table view listing all of the dining halls in the database. Simply tap one to "drill down" and get a list of the "restaurants" (if any) inside that dining hall. Tapping on one of those will give you a list of foods available there. Tap on a food to bring up a screen showing the calories.

Food detail view

Here you can see the detailed nutrition facts from VT's web sites. Simply tap the button to add this food to today's list of foods.
